# 社区行为准则

为了让 `WeChat_Agent` 社区保持开放、友好、专业，参与者需共同遵守以下准则。

## 我们鼓励的行为

- 使用尊重、包容、建设性的表达方式
- 关注问题本身，给出可执行的反馈
- 尊重不同技术背景、经验和观点
- 在讨论中保持耐心，帮助新贡献者成长

## 我们不接受的行为

- 人身攻击、侮辱、歧视、骚扰或威胁
- 恶意争吵、挑衅、刷屏、钓鱼行为
- 泄露他人隐私信息或敏感数据
- 任何违反法律法规或平台规则的内容

## 适用范围

本准则适用于本仓库相关的公开沟通场景，包括但不限于：

- Issues
- Pull Requests
- Discussions
- 其他与项目维护相关的协作渠道

## 违规处理

维护者有权对不当行为采取必要措施，包括：

- 提醒与警告
- 隐藏或删除不当内容
- 临时或永久限制参与权限

## 联系方式

如遇到骚扰、不当行为或其他社区安全问题，请通过私下方式联系维护者处理。
